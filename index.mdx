## Overview

CrudPark is a modern parking management system built with ASP.NET Core 9.0 that provides a robust API for managing parking operations. From membership tracking to real-time stay monitoring and comprehensive reporting, CrudPark streamlines parking facility operations with a clean, well-architected solution.

## Technology Stack

CrudPark is built with modern technologies and follows clean architecture principles:

- **ASP.NET Core 9.0** - High-performance web framework
- **Entity Framework Core** - ORM with PostgreSQL support
- **PostgreSQL** - Reliable relational database
- **Swagger/OpenAPI** - Interactive API documentation
- **Clean Architecture** - Layered separation of concerns (API, Application, Core, Infrastructure)
- **Docker** - Containerized deployment support
